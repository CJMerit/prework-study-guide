Prework Study Guide

HTML

  • HTML elements build the structure of the webpage to provide a framework for navigation.
  • Commonly used elements of HTML are Text, Image, Header, and List elements.
  • Elements for page division and description are Meta-related, Content division, and Semantic elements.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • There are three ways to use CSS: inline, internal style sheet, and external style sheet.
  • Inline CSS is used to style a single element, within the code of that element.
  • Internal style sheet CSS is an entire stylesheet embedded in the head element.
  • External style sheet CSS is a separate stylesheet that is linked to the code.

Git

  • All Git commands in CLI must begin with git.
  • Use "git checkout" to switch branches, "git checkout -b" to create a new branch.
  • Use "git status" to check current branch.
